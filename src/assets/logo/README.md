# Logo de empresa

Coloca aquí el logo de la empresa con el nombre `logo.png` (o `.svg`).

- **logo.png** — versión principal (recomendado: fondo transparente, al menos 256×256px)
- **logo-mark.png** — versión icono cuadrado/circular para el modo mini del sidebar (recomendado: 64×64px)

Una vez colocados, actualiza la variable `LOGO_SRC` en `src/components/layout/Sidebar.tsx`.
